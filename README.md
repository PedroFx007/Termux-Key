# Termux-Key
Add extra keys for your termux terminal!

# How to use

apt update -y && apt upgrade -y

apt install git -y

git clone https://github.com/PedroFx007/Termux-Key

cd Termux-Key

chmod +x keys.sh

bash keys.sh

Now select number 1 and enjoy ;)
